Images

The Wizzyshop API provides a specific endpoint for treating images.

Endpoint: http://wizzyshop.eddmi.com/api/image/index

This endpoint will return the requested image with the requested size in PNG format. If no size is passed, it will return an image with 10px height and the original width.

The API users relative URLs returned for each request, when are images available.

Examples

Brands 


"image": "brand/8/image.png",
"cover_image": "brand/8/cover_image.png",

Products 


"photos": [
       "product/30/1.png",
       "product/30/2.png"
]

Promotions 


"image": "promotion/2/image.png"

In order to keep the image aspect ratio, it will be used the biggest value passed between the height and width.


Request Examples

Request 

GET http://wizzyshop.eddmi.com/api/image/index?url=brand/26/1.png&height=100
Reponse 
An PNG image with 100px height and the width to keep the original aspect ratio

Request 

GET http://wizzyshop.eddmi.com/api/image/index?url=brand/26/1.png&width=100
Reponse 
An PNG image with 100px width and the height to keep the original aspect ratio
